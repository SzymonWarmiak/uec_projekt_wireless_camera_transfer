# Servo Wiring

Servo control is generated by `basys_cam`, not `basys_station`.

Connect the servo signal wire to `JXADC1` / `XA1_P` on the Basys Cam board.
In the XDC this is:

```tcl
set_property PACKAGE_PIN J3 [get_ports servo_pwm]
```

Wiring:

- Servo signal: `JXADC1` / `XA1_P` / FPGA pin `J3`
- Servo ground: `JXADC5` or `JXADC11` ground
- Servo power: external 5 V supply is recommended; connect its ground to Basys ground

Optional `esp_cam_servo` command inputs:

- `esp_cam_servo` `GPIO17` / `D17` -> `JXADC2` / `XA2_P` / FPGA pin `L3` for left
- `esp_cam_servo` `GPIO16` / `D16` -> `JXADC3` / `XA3_P` / FPGA pin `M2` for right
- `esp_cam_servo` `GND` -> `JXADC5` or `JXADC11` ground

Do not power a larger servo directly from the Basys 3.3 V pin. The FPGA output is only the PWM control signal.
