# Jezdzik - pliki do pobrania

## Android

Pobierz i zainstaluj:

```text
Jezdzik.apk
```

Telefon moze zapytac o zgode na instalacje aplikacji spoza sklepu.

## Windows

Pobierz:

```text
Jezdzik_Windows.zip
```

Rozpakuj ZIP i uruchom:

```text
jezdzik.exe
```

Nie przenos samego pliku `.exe` bez folderu `data` i bibliotek `.dll`, bo aplikacja Windows ich potrzebuje.
